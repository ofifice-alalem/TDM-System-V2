# TDM-System-V2
نظام إدارة التوزيع والمبيعات - الإصدار الثاني

## حول المشروع
هذا مشروع Laravel لإدارة نظام التوزيع والمبيعات (TDM - Trade Distribution Management).

## متطلبات النظام
- PHP >= 8.2
- Composer
- Node.js & NPM
- قاعدة بيانات (MySQL/SQLite)

## التثبيت
1. تم تثبيت Laravel بنجاح
2. قم بتشغيل الخادم المحلي:
   ```bash
   php artisan serve
   ```

## الميزات
- إدارة المسوقين والمتاجر
- نظام الطلبات والمبيعات
- إدارة المخزون
- تتبع الأرباح والعمولات
- نظام التقارير
