===========================================
        الوضع الداكن - Dark Mode
===========================================

## 🌓 التفعيل

### Tailwind Config
```javascript
// tailwind.config.js
module.exports = {
  darkMode: 'class', // أو 'media' للتبديل التلقائي
  // ... باقي الإعدادات
}
```

### HTML Setup
```html
<!-- إضافة class="dark" للتفعيل -->
<html dir="rtl" lang="ar" class="dark">
```

### Toggle Button
```html
<button @click="darkMode = !darkMode; 
                localStorage.theme = darkMode ? 'dark' : 'light';
                document.documentElement.classList.toggle('dark')"
        class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800">
    <!-- أيقونة الشمس (Light Mode) -->
    <svg x-show="!darkMode" class="w-5 h-5" fill="none" viewBox="0 0 24 24" stroke-width="1.5" stroke="currentColor">
        <path stroke-linecap="round" stroke-linejoin="round" d="M12 3v2.25m6.364.386l-1.591 1.591M21 12h-2.25m-.386 6.364l-1.591-1.591M12 18.75V21m-4.773-4.227l-1.591 1.591M5.25 12H3m4.227-4.773L5.636 5.636M15.75 12a3.75 3.75 0 11-7.5 0 3.75 3.75 0 017.5 0z" />
    </svg>
    <!-- أيقونة القمر (Dark Mode) -->
    <svg x-show="darkMode" class="w-5 h-5" fill="none" viewBox="0 0 24 24" stroke-width="1.5" stroke="currentColor">
        <path stroke-linecap="round" stroke-linejoin="round" d="M21.752 15.002A9.718 9.718 0 0118 15.75c-5.385 0-9.75-4.365-9.75-9.75 0-1.33.266-2.597.748-3.752A9.753 9.753 0 003 11.25C3 16.635 7.365 21 12.75 21a9.753 9.753 0 009.002-5.998z" />
    </svg>
</button>

<script>
// تحميل الإعداد المحفوظ
if (localStorage.theme === 'dark' || (!('theme' in localStorage) && window.matchMedia('(prefers-color-scheme: dark)').matches)) {
    document.documentElement.classList.add('dark')
} else {
    document.documentElement.classList.remove('dark')
}
</script>
```

## 🎨 نظام الألوان

### الخلفيات
```html

<body class="bg-gray-100 dark:bg-gray-900">

<!-- خلفية البطاقات -->
<div class="bg-white dark:bg-gray-800">

<!-- خلفية ثانوية -->
<div class="bg-gray-50 dark:bg-gray-700">

<!-- خلفية Sidebar -->
<aside class="bg-gray-800 dark:bg-gray-950">
```

```html

<!-- نص ثانوي -->
<p class="text-gray-700 dark:text-gray-300">
```

```html

               text-gray-700 dark:text-gray-200">

<!-- زر خطر -->
<button class="bg-red-600 hover:bg-red-700 dark:bg-red-500 dark:hover:bg-red-600 text-white">
```

### الحقول
```html
<input class="bg-white dark:bg-gray-800 
              border-gray-300 dark:border-gray-600 
              text-gray-900 dark:text-gray-100
              focus:ring-blue-500 dark:focus:ring-blue-400">
```

```html
    <p class="text-red-700 dark:text-red-300">
```

## 🪟 Modals في Dark Mode

```html
<!-- Overlay -->
<div class="bg-gray-900 dark:bg-black bg-opacity-50 dark:bg-opacity-70">

<!-- Modal Container -->
<div class="bg-white dark:bg-gray-800 rounded-lg shadow-xl dark:shadow-gray-900/80">
    
    <!-- Header -->
    <h3 class="text-gray-900 dark:text-gray-100">
    
    <!-- Close Button -->
    <button class="text-gray-400 dark:text-gray-500 
                   hover:text-gray-600 dark:hover:text-gray-300">
    
    <!-- Body Text -->
    <p class="text-gray-700 dark:text-gray-300">
    
    <!-- Info Box -->
    <div class="bg-gray-50 dark:bg-gray-700/50 rounded-lg p-4">
        <span class="text-gray-600 dark:text-gray-400">
        <span class="text-gray-900 dark:text-gray-100">
    </div>
    
    <!-- Input Field -->
    <input class="bg-white dark:bg-gray-900 
                  border-gray-300 dark:border-gray-600 
                  text-gray-900 dark:text-gray-100">
    
    <!-- Buttons -->
    <button class="bg-gray-200 dark:bg-gray-700 
                   hover:bg-gray-300 dark:hover:bg-gray-600 
                   text-gray-700 dark:text-gray-200">
</div>
```

## 🎯 أفضل الممارسات

1. **استخدم dark: prefix** لكل class يحتاج تغيير في Dark Mode
2. **الألوان الملونة** (أزرق، أخضر، أحمر) تبقى واضحة في الوضعين
3. **الرمادي** يحتاج عكس (gray-100 → gray-800)
4. **الشفافية** استخدم `/20` أو `/30` للخلفيات الملونة في Dark Mode
5. **الظلال** استخدم `shadow-gray-900/50` في Dark Mode
6. **الحدود** استخدم `border-gray-700` بدلاً من `border-gray-200`

```html
            </p>
```
